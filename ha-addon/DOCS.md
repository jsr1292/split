# Split Expense Tracker

Self-hosted expense sharing app. Split bills with friends and family.

## Features
- Multi-currency support with live exchange rates
- Debt simplification (minimum transfers)
- Offline support (PWA)
- Multiple themes
- Recurring expenses
- Item-level splitting

## Setup
1. Open the app via Ingress or direct port
2. Create an account
3. Create a group and start adding expenses
